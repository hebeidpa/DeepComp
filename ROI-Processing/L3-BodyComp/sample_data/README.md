# Sample data

This folder is reserved for fully anonymized sample NIfTI data used to reproduce the VoxTell-L3 body composition module.

The test NIfTI file is not included in the standard GitHub repository because medical image files are often large and must be carefully anonymized before public sharing.

If a sample NIfTI is shared, it should be provided through Git LFS, GitHub Release, Zenodo, OSF, or an institutional repository.

Expected output for the internal test case is available in:

~~~text
VoxTell-L3-BodyComp/examples/expected_output.json
~~~
