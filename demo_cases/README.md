# Demo cases

This folder contains two anonymized demonstration cases for DeepComp.

Each case includes:
- `plain_CT.nii.gz`: non-contrast CT image
- `plain_L3_bodycomp_label.nii.gz`: L3-level body composition label on non-contrast CT
- `venous_CT.nii.gz`: venous-phase CT image
- `venous_tumor_peritumor_label.nii.gz`: tumor and peritumoral label on venous-phase CT

The files have been anonymized and renamed before public release. Patient-level identifiers, original file names, and clinical metadata are not included.
